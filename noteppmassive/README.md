# noteppmassive - V++ Massive Bubble IBP Notes

## Overview
This folder contains LaTeX notes and documentation for the **V++ massive bubble** integration-by-parts (IBP) system in de Sitter space.

## Files
- `main.tex` - Complete LaTeX notes for V++ massive bubble IBP system.
- `main.pdf` - Compiled PDF version.

## Workflow
1.  **Edit**: Modify `main.tex`.
2.  **Compile**: Use `latex-compiler` skill (`pdflatex -interaction=nonstopmode main.tex`).
3.  **Verify**: Check PDF for overflow/references.
