# 微分方程管理（DE）

本技能用于从 Kira 约化结果构建微分方程矩阵，并完成主积分选择、编号映射与一致性检查。

## 适用场景
- 用户要求从 Kira 结果构建微分方程
- 用户需要主积分编号截断或调整基底
- 用户需要检查约化后除主机分外余项为零和微分方程标度关系式

## 输入与依赖
- `result/`：编号映射、参数、基底等复用文件
- `results_wsl/`：Kira 约化规则与主积分列表
- 约化规则需可反复应用直到稳定

## 主积分处理
- 从 `masters.final` 读取主积分编号并显示供用户参考
- 若主积分编号出现“断崖式跃迁”，默认认为后续编号不是主积分
- 提供可手动设置的截断编号参数
- 支持基底在 Kira 编号形式与指标形式之间互相转换

## 关键能力
- 构建动力学量偏导算子（如对外参/尺度的偏导）
- 将偏导结果通过约化规则投影到主积分基底，输出余项积分指标形式
- 输出并保存微分方程矩阵

## 输出与检查
- 保存 DE 矩阵到文件
- 输出余项积分指标形式、缩放关系检查结果
- 发现不一致需回溯修正，存在余项时需使用`ibp-manager`试探出合理的ibp指标范围
